TCH Forecaster Installation

The TCH Forecasting system is made up of three major pieces:

  • TCH Forecaster: can be built into two pieces, the first is a web application that was originally used to validate the forecaster, the second part is a java jar that can be included into java applications. Requires MySQL to run as web application.
  • TCH Forecast Tester: does not directly include the forecaster but is used instead to test the TCH Forecaster, it connects to the TCH Forecaster when it runs as the web application and to Web1. Requires MySQL to run as web application.
  • TCH ImmSys Web1: immunization information system that tracks TCH immunizations and produces forecasts. The forecaster is included as a jar and Web1 uses it to forecast for patients. Requires Oracle database to deploy.

TCH Forecaster

Installation Instructions

  1. Download and install MySQL, Ant, and Tomcat
  2. Download the TCH Forecaster project to local system.
  3. Run the latest {project}/db/tchforec_v*.sql file into MySQL. Example command: mysql -uusername -ppassword < tchforec_v*.sql
  4. Create user in MySQL to access. Example is given in the top of {project}/db/start.sql
  5. Edit {project}/web/WEB-INF/classes/dbpool.properties file to indicate the proper URL, username and password for the new MySQL database.
  6. Run ant deploy "-Dtomcat.home={location of Tomcat}"
  7. Startup Tomcat
  8. Go to http://localhost:8080/fv/fv and login as "Unknown User"
  9. Queries can be sent to http://localhost:8080/fv/forecast
