TCH Forecaster for Web

This is a temporary documentation page to describe how to download and run the Tomcat version of the TCH Forecaster ready to receive requests from RPMS.

Installation Instructions

  • Install latest version of Java Development Kit JDK
  • Install Apache's Tomcat
  • Save fweb.war to the Apache Tomcat webapps folder.
  • If Tomcat is running the war should deploy and the forecaster should be operational and listening on port 6708 within the next 10 seconds.

How to Test

On a windows system run the following command:

telnet localhost 6708

You will get a blank screen. Copy and past the input string and press enter.

The results will be sent back and the connection closed.
