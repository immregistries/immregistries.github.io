Overview

This page provides documentation for DQA in version 1.0. In this version the DQA is best maintained, deployed and configured by a resource with a high level of technical ability. Later versions of the DQA will be distributed in versions that are easier to administer and eventually will be easy for non-technical resources to manage and deploy.

Project Layout

The project is now in four major pieces:

  • Database https://ois-dqa.svn.sourceforge.net/svnroot/ois-dqa/database/trunk
  • Core https://ois-dqa.svn.sourceforge.net/svnroot/ois-dqa/core/trunk
  • Web Service https://ois-dqa.svn.sourceforge.net/svnroot/ois-dqa/webservice/trunk
  • Web UIhttps://ois-dqa.svn.sourceforge.net/svnroot/ois-dqa/webui/trunk
