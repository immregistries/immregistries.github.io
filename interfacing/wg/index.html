Presentation Recordings

The presentations on Wednesday morning were given to the work group in Decatur and were also shared using GoToMeeting. During the meeting we recorded the presentations on video and made a separate audio recording. Half way through the presentations a technical issue closed the call so the presentation was interrupted.

The video presentations require installation of the GoToMeeting codec. This should be installed automatically if you have recently used GoToMeeting. If you have not, or would like to ensure you have the codec, instructions are here.

Please download and save the files only once, in order to save on server bandwidth. Also, please do not share this link with others only external expert panel members are being asked to review and comment.
